A simple automation tool built with Bash and Python to monitor website availability and analyze system logs.

### Features:
- **Bash Monitoring**: `check_site.sh` tracks HTTP status codes.
- **Log Analysis**: Python script `analyzer.py` scans logs for errors and generates reports.
- **Automation**: `run_all.sh` executes the entire pipeline.

### How to use:
1. Navigate to: `cd log_monitor`
2. Run script: `./run_all.sh`
3. Results in: `final_report.txt`
